Examples
--------

List emojis for a resource::

   emojis = obj.awardemojis.list(get_all=True)

Get a single emoji::

   emoji = obj.awardemojis.get(emoji_id)

Add (create) an emoji::

   emoji = obj.awardemojis.create({'name': 'tractor'})

Delete an emoji::

   emoji.delete
   # or
   obj.awardemojis.delete(emoji_id)
